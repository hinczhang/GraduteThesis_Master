### 后续计划注意事项
有两个点可以注意：  
****  
**如何进行chunking:**  
1. 一种是定量法，如fisher度量进行的距离阈值划分；  
2. 一种是定性法，如根据功能区块进行划分（如教室区，教授办公区等）。  
****
**chunking的可视化方法:**  
1. 一种是块内描述，对于块内事务进行某种可视化联系，如色块、形状等；  
2. 一种是块间描述，对于每个chunk进行某种可视化，主要是从更大的角度让用户感知到当前的空间结构。
